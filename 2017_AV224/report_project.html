Project - Survey of contemporary topics in Computer Organization and Operating Systems

  • The report has to be written in LaTeX using this template.

  • The report needs to be 10 pages long, with 10 pt font, and single line spacing.

  • The report needs to include a short abstract as well as properly formatted references (preferably using Bibtex).

  • You have to choose two topics each out of Computer Organization and Operating Systems.

  • Some topics are given below. You can choose out of the given topics or write about topics that you have picked but with the consent of the instructor.

Computer Organization Topics

1 MIPS instruction set architecture
2 ARMv7 instruction set architecture
3 SIMD extensions
4 Case study of Intel Core i7 pipeline
5 Cache coherence mechanisms
6 RAID
7 SCSI
8 USB protocol
9 Virtual machines
10 GPU and GPU computing
11 Content distribution networks
12 The architecture of your mobile phone
13 Data centers - computation and networking
14 Amazon’s EC2 - how does it work?

Operating Systems Topics

1 Compilers, Linkers, Loaders and Executable file formats
2 Deadlock detection and recovery schemes
3 Filesystems - FAT, EXT234, NFS
4 Loadable kernel modules in Linux
5 Case study of Linux - process management and scheduling
6 Network stacks in the operating system & TCP/IP
7 Case study of Windows 7 - process management and scheduling
8 Case study of Mach - process management and scheduling
9 POSIX
10 Blockchains
11 Pthread library and its use
12 Monitors for process synchronization
13 Malware and Botnets
14 Case study of Android operating system
